
  После загрузки программы , отключите вход Init процессора от цепи GND , затем выключите и
 включите питание процессора.

  1. Запустите программу DspHost.
  2. Нажмите клавишу 'Enter', в ответ должна появляться новая строка с
  символом

  >
   Если символ '>' не появляется , нажмите клавишу 'F10' и проверьте
 установки COM порта.
  Должно быть : COM1 ( или COM2)
                115200 Baud
                8 Data Bits
                2 Stop Bits
                No Parity ( без птички).

      Повторите п.1.


  3. Введите команду
  ID
  В ответ отобразится

  ID 0 nnn mmm

  nnn и mmm - шестнадцатеричные числа.
  Их нужно записать и сообщить в Факом.

   В ответ будет прислан код  СССС.

  После получения кода CCCC его нужно ввести

  ID CCCC

   затем нажмите клавишу 'Enter'
  Должно отобразиться

 ID CCCC == ID OK


 4.  В левом верхнем углу выберите меню 'File' и в нем функцию 'Open'.
    Выберите файл  TST081112.eee , указав путь к нему. Затем нажмите мышкой
    кнопку 'Открыть'. Окно выбора исчезнет.

 5.  Нажмите клавишу 'F4' - загузите выбранный файл.
     На дисплее должно появиться
     >VL
     >

     Если '>' не появилось, необходимо несколько раз нажать клавишу
  'Enter' до появления символа > на новой строке.
     Затем повторить п.5 .

 6. Затем введите команду

 WE

     затем 'Enter'.

     Контроллер сохранит данные в EEPROM.
